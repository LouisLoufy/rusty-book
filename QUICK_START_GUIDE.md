# LoongBot Docs - Quick Start Guide 🚀

## Welcome to the Redesigned Documentation!

Your documentation has been completely transformed with cutting-edge 3D effects and interactive features. Here's how to explore everything:

---

## 🎯 Key Features & How to Use Them

### 1. 3D Sidebar Navigation
**Location:** Left side of the page

**What it does:**
- Beautiful glassmorphism cards
- 3D tilt effect that follows your mouse
- Gradient-highlighted active links

**How to use:**
- Simply move your mouse over the sidebar sections
- Watch them tilt and float in 3D space
- Click any link to navigate

---

### 2. Floating TOC Orb (Table of Contents)
**Location:** Bottom-right corner (floating orb with breathing animation)

**What it does:**
- Quick navigation to any section on the current page
- Beautiful radial menu expansion
- Shows active section in real-time

**How to use:**
1. Click the circular orb in the bottom-right
2. A radial menu will expand with all headings
3. Hover over any item to see the full title
4. Click to jump to that section
5. Click outside to close

**Mobile:** On mobile devices, a panel will appear instead of the radial menu

---

### 3. Theme Selector
**Location:** Top-right corner (colorful button next to menu)

**What it does:**
- Switch between 4 beautiful gradient themes
- Instant color transitions
- Saves your preference automatically

**Available Themes:**
- 🟣 **Purple-Pink** (Default) - Vibrant and modern
- 🔵 **Blue-Green** - Cool and professional
- 🟠 **Orange-Red** - Warm and energetic
- 🌈 **Aurora** - Multi-color gradient

**How to use:**
1. Click the theme button (colorful square icon)
2. Choose your favorite theme from the preview cards
3. Watch the smooth color transition
4. Your choice is saved for next visit!

---

### 4. Interactive Code Playground
**Location:** Throughout documentation (code blocks)

**What it does:**
- Edit and run JavaScript code directly
- See real-time output in console
- No need to leave the documentation

**How to use:**
1. Find a code block in the documentation
2. Edit the code as you like
3. Click **"Run"** button or press `Ctrl+Enter` (Mac: `Cmd+Enter`)
4. See the output in the console below
5. Try `console.log()`, `console.error()`, `console.warn()`
6. Click **"Reset"** to restore original code
7. Click **"Share"** to copy code to clipboard

**Example:**
```javascript
console.log("Hello, LoongBot!");
const sum = 5 + 3;
console.log("5 + 3 =", sum);
```

---

### 5. AI Smart Assistant
**Location:** Bottom-right corner (orb below the TOC, with sparkle icon)

**What it does:**
- Intelligent search across all documentation
- Quick access to common questions
- Fast fuzzy matching

**How to use:**
1. Click the AI orb (sparkle icon ✨)
2. Type your question or keyword
3. Results appear instantly as you type
4. Click a result to navigate to that page
5. Try the quick question chips for common queries

**Quick Questions:**
- "How to install LoongBot?"
- "Getting started guide"
- "API documentation"
- "Configuration options"

---

### 6. Annotation System (Note-Taking)
**Location:** Right side panel (appears when you create notes)

**What it does:**
- Highlight and annotate any text
- Add your own notes and comments
- Notes are saved locally in your browser

**How to use:**
1. **Select any text** in the documentation
2. Click **"Add Note"** button that appears above
3. Type your note in the dialog
4. Click **"Save Note"**
5. Your annotation is highlighted in the text
6. View all notes in the right panel
7. Click **X** to delete a note

**Notes:**
- Notes are saved per page
- They persist across browser sessions
- Private to your browser (not shared)

## 🔧 Troubleshooting

### Performance Issues?
- Try disabling browser extensions
- Close other tabs to free up resources
- Use a modern browser (Chrome, Firefox, Safari, Edge)

### 3D Effects Not Working?
- Make sure your browser supports CSS 3D transforms
- Update to the latest browser version
- Check if "Reduce Motion" is disabled in system settings

### Themes Not Saving?
- Ensure cookies/localStorage is enabled
- Check browser privacy settings
- Try clearing browser cache and reload
